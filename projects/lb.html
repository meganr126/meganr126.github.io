LawBook

The easiest way to learn about your local laws.

About

This project was completed for Info 4240: Designing Technology for Social Impact at Cornell. The assignment for this project was to craft a technology that promotes debate and democracy. My solution was a voter information app called LawBook.

When the user loads the app, LawBook asks for a location; this allows LawBook to immediately give accurate, relevant information. That said, the user can always choose to pick another state. Using a resizeable map of the United States, the user can select any state they would like to learn about.

On the page for a given state, a lot of information is displayed. Users can learn about how to register to vote in their state, who is currently in office, what local laws and ordinances are in place, among many other things. In all, LawBook is a one-stop shop for any legal- and election-related questions you might have.

The user experience is designed to be simple and vibrant. On a state's page, individual topics can be scrolled through and opened; this allows for a simple opening view to avoid the user being hit with too much information at once.
